# Gym Xiangqi
Gym Xiangqi is a reinforcement learning environment of Xiangqi, Chinese Chess, game.
The project is built on top of a popular reinforcement learning framework called OpenAI Gym.

## Getting Started
Installing and using Gym Xiangqi easy. If you have experience working with other OpenAI Gym 
environments you are already off to a good start. We also welcome you to checkout our 
[documentation page](https://gym-xiangqi.readthedocs.io/en/latest/).

### Installation
First, clone the repository - `git clone https://github.com/tanliyon/gym-xiangqi.git`

Enter the repository - `cd gym-xiangqi`

Install the core dependencies - `pip3 install -e .`

(optional) Install the dependencies for development - `pip3 install -r requirements.txt`
